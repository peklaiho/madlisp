# MadLisp

MadLisp is a [Lisp](https://en.wikipedia.org/wiki/Lisp_%28programming_language%29) interpreter written in PHP.

## Requirements

The project requires PHP 7.4 or newer and [Composer](https://getcomposer.org/).
